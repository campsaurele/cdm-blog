# World Cup Blog

## What for ?

DWWM exercice to manipulate PHP, DB and CRUD method. I also want to use this project as an opportunity to build a well-structured application with a meaningful directory structure.

This exercise takes place in localhost, day after day we keep adding new features such as CRUD, creating and renaming images, login etc..

I'm actually trying to make CRUD function adaptating to the DB. Instead of writing every SQL Query, table, plaholder and value..

I'd like to stock the value of my $\_POST in a var that would be transform and filter into a string to add it into a $SQLquery used in the preparation and the execution of the query. This way, I could make my CRUD automatic and adapting to any DB, or just make my actual DB scalable by adding tables.

Next Hard step would also to generate automatically forms (in create and update) by analysing the type of data in the table and adjusting input about it.
